# Phemex futures place trades 🌟 🐟 🐻
Application for entering pending futures orders from Excel on the Phemex exchange.

## How to build
```bash
./build.sh
```

## How to deploy
```
./deploy.sh
```

## How to prepare
```bash
./prepare.sh
```

## How to run
```bash
./run.sh
```

## Technologies
* Python 3
* Pandas
* CCXT
